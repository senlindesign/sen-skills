# i-write - Personal Writing System

Language: English | [中文](README.zh-CN.md)

Make AI write in a way that feels like you wrote it yourself.

Core principle: **understand the writer first, then write.**

## What Public Writing Is Made Of

Any piece of public writing can be split into four parts:

1. **Frame** - the judgment the writer feels they have to make. It grows out of their values and way of seeing the world → `frame.md`
2. **Style** - the writer's voice: sentence rhythm, word choices, opening habits, ending habits → `style.md`
3. **Craft** - how the argument is cut, arranged, and sharpened. This is shared writing technique → `craft.md`
4. **Medium** - how the same piece changes when it appears on WeChat, Xiaohongshu, LinkedIn, Medium.com, or another medium → `medium.md`

The first two belong to the writer. The last two are shared tools and rules. This skill puts all four back together into a finished piece.

## How It Works

**First use: Deep Interview** (`interview.md`)

Before writing, the skill first learns the writer. The interview extracts two things:

- Values, beliefs, and ways of seeing the world. These are tested through cross-domain questions and work retrospectives, not taken from self-summary → generates `profile/frame.md`
- Existing writing preferences, sentence patterns, taste, and hard limits → generates `profile/style.md`

The interview has three lengths: short (20 questions), medium (50 questions), and long (100 questions).

**Every writing session after that:**

```
spoken ideas from the user
   ↓
frame.md   stabilizes the point of view
   ↓
confirm the argument skeleton before drafting
   ↓
style.md   applies the writer's voice
   ↓
craft.md   sharpens the writing
   ↓
medium.md  adapts the piece to each medium
   ↓
one finished markdown file per medium
```

## Common Prompts

- "Write a piece about XX" - starts the writing flow. If no profile exists, the skill starts with the interview.
- "i-write" - invokes the skill directly.
- "Redo the interview" or "update my style" - updates the profile incrementally.
- "Adapt this for Xiaohongshu and WeChat" - produces multiple medium-specific versions.
